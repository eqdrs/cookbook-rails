# Cookbook

CookBook built with Ruby on Rails, made by me.

### Run the project

Download the project:

`git clone git@github.com:eqdrs/cookbook-rails.git`

Install the dependencies:

`bundle install`

Run the server:

`bundle exec rails s`

### Run tests

`bundle exec rspec`

### Dependencies

* Ruby 2.6.0
* Rails 5.2.2
